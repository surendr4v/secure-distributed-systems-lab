# Release Notes

## v0.1.0 (2026-02-28)
- Reframed project as secure financial transaction microservice with zero-trust defaults.
- Added JWT + mesh-token guard, RBAC decorator/guard, and audit logging interceptor.
- Introduced strict config validation and optional mTLS listener.
- Added transaction domain skeleton with validated DTOs and in-memory ledger.
- Provided Dockerfile, sample env, and initial unit tests.
- Documented architecture, threat model, and security feature set in README.
